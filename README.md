# 🚀 TaskOrbit – AI Powered Team Task Management Platform

TaskOrbit is a futuristic AI-powered task management and collaboration platform designed for modern teams.  
It combines intelligent productivity workflows, real-time collaboration, Orbit visualization, and premium UI/UX into a single powerful workspace.

---

# 🌌 Features

## ✨ Core Features
- AI-powered productivity dashboard
- Modern responsive UI
- Orbit View task visualization
- Kanban task management
- Project management system
- Smart analytics dashboard
- Focus Mode for deep work
- Command Palette (Ctrl + K)
- Notification Center
- Dark / Light Theme
- Drag and Drop Tasks
- Interactive Charts & Insights
- Persistent LocalStorage-based data
- Fully frontend-based architecture

# 🛰️ Orbit View (USP)

The Orbit View is the signature feature of TaskOrbit.

Features include:
- Animated orbiting task nodes
- Dynamic motion effects
- Priority-based glow colors
- Interactive task clusters
- Futuristic visualization experience

Priority Indicators:
- 🔴 High Priority → Red Glow
- 🔵 Medium Priority → Blue Glow
- ⚪ Low Priority → Gray Glow
- ✅ Completed → Transparent/Faded

# 🖥️ Run Locally

## Prerequisites

Make sure you have installed:

- Node.js (Latest Recommended)
- npm or yarn

Verify installation:

```bash
node -v
npm -v
```

---

# 📦 Installation

Install dependencies:

```bash
npm install
```

---

# 🔑 Environment Variables

Create a `.env.local` file in the root directory.

Add your Gemini API Key:

```env
GEMINI_API_KEY=your_api_key_here
```

---

# ▶️ Start Development Server

Run the app locally:

```bash
npm run dev
```

The application will start on:

```bash
http://localhost:5173
```

---

# 🏗️ Build for Production

Generate production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
